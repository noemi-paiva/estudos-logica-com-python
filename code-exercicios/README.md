*** Variáveis e Atribuição de Valores em Python ***

Em Python, variáveis são usadas para armazenar valores que podem ser utilizados ao longo do programa.
Para criar uma variável, basta escolher um nome e usar o sinal de igual (=) para atribuir um valor a ela.

Exemplo:
nome = "Alice"
idade = 20


Neste exemplo:

- nome guarda um texto (string)
- idade guarda um número inteiro (int)

Regras importantes:

- O nome da variável não pode ter espaços
- Não pode começar com número
- Use nomes claros e em minúsculo

O Python diferencia letras maiúsculas e minúsculas

Exemplo válido:
nota_final = 8

Exemplo inválido:
2nota = 8


Tipos básicos de valores:

- int → números inteiros (10, 25)
- float → números com vírgula (7.5, 3.14)
- str → textos ("Olá", "Python")
- bool → verdadeiro ou falso (True, False)


Entrada de dados do usuário:

Para receber valores digitados pelo usuário, usamos a função input():
nome = input("Digite seu nome: ")


Quando o valor digitado precisa ser número, usamos conversão:
idade = int(input("Digite sua idade: "))

Resumo:

Variáveis permitem guardar informações e reutilizá-las no programa, tornando o código mais organizado e fácil de entender.

Esse é um dos primeiros passos para aprender programação em Python!
